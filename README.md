# L3n0m's lab - blog técnico e portifólio pessoal

Este repositório representa o código-fonte do meu blog pessoal e portfólio público de análise de tecnologia e segurança da informação.

 ## Objetivo

Este blog tem como seu objetivo:

- Manter um registro técnico e pessoal de métodos, processos, experimentos e conquistas, visando evolução e aprendizado contínuo
- Aprimorar capacidades de estruturação e formulação de relatórios e análises técnicas
- Compartilhar meu progresso pessoal na área com colegas, entusiastas de tecnologia e possíveis empregadores
- Servir como portfólio público para oportunidades acadêmicas e profissionais

 ## Metodologia

Atualmente são utilizadas as seguintes ferramentas para a estruturação do blog:

- Markdown
- Git / GitHub
- Gerador de site estático (a definir)

 ## Status

 Em estágio inicial de desenvolvimento - estabelecimento de conceito e estruturação básica.

 ## Observações

Este projeto faz parte do meu processo de aprendizado e evolução técnica. Melhorias, ajustes e refatorações são esperadas ao longo do tempo.
